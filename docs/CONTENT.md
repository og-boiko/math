## Принципи генераторів

1. Уникаємо тривіальних (`5 + 0`, `1 × x`).
2. Числа без зайвої складності (ділення — ціле).
3. Завжди ≥ 2 рівні підказок + рішення.
4. Кожен генератор має fallback при некоректних параметрах (рекурсія на простіший).
5. Підтримка нормалізації відповіді (кома/крапка, пробіли).
