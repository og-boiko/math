# MathQuest — Технічне завдання

## 1. Огляд

**MathQuest** — веб-застосунок (PWA) для тренування математики для Артема (10 р.), який закінчив 4 клас за програмою «Інтелект». Мета — повторити вивчене і підготуватись до 5 класу через гейміфіковані щоденні тренування.

## 2. Цілі та не-цілі

### Цілі

- Щоденна практика 15–20 хв без примусу
- Покрити **усі ключові теми** 4 класу + місток у 5 клас
- Автоматично повертати помилки для опрацювання
- Дати батькам видимість прогресу

### НЕ-цілі (поки що)

- Мультиплеер / соц-механіки
- Платні підписки / магазин
- Власна авторизація через email/пароль
- Озвучка / читання задач уголос

## 3. Користувачі

### Артем (10 років) — основний

- Любить: космос, динозаврів, країни, цікаві факти, Roblox, Minecraft
- Користується переважно з телефона
- Навчальний рівень: 4 клас «Інтелект»

### Батько — вторинний

- Заходить у батьківську панель раз на 1–3 дні
- Хоче бачити: над чим працює син, де слабкі місця, скільки часу витрачає
- Налаштовує: щоденну ціль, доступні теми, складність

## 4. Тематичні сетинги (themes)

При першому запуску дитина обирає **сетинг**, який змінює візуал/назви/локації.

| ID       | Назва             | Аватар             | Локації                    | Бос                  | Валюта    |
| -------- | ----------------- | ------------------ | -------------------------- | -------------------- | --------- |
| `space`  | Космос            | Астронавт          | Планети                    | Чорна діра           | Кристали  |
| `dino`   | Динозаври         | Палеонтолог        | Епохи                      | Тиранозавр           | Бурштин   |
| `world`  | Подорож світом    | Мандрівник         | Континенти                 | Бермудський трикутник| Монети    |
| `voxel`  | Воксельний світ   | Будівельник        | Біоми                      | Ендер-дракон         | Смарагди  |
| `blocky` | Блочний світ      | Аватар-блок        | Обби-світи                 | Бос-обба             | Зірочки   |

> Не використовуємо офіційні текстури/логотипи. Робимо «inspired by».

## 5. Функціональні вимоги

### 5.1. Онбординг

- REQ-001: Привітання з ім'ям, вибір сетингу.
- REQ-002: Тур із 3 екранів.
- REQ-003: Запит установки PWA.
- REQ-004: Створення профілю в localStorage.

### 5.2. Головний екран (Hub)

- Картка «Щоденне завдання» з прогресом.
- Картка «Вільне тренування».
- Картка «Робота над помилками» (якщо є черга).
- Кнопка «Карта світу».
- Іконка профілю + бейдж стріку.

### 5.3. Щоденне завдання

Структура: розминка (5 усного) → тема дня (10 задач) → міні-бос → нагорода.

### 5.4. Вільне тренування

10 тем, статус: 🔒/⭐/👑. Сесія 10 задач.

### 5.5. Екран задачі

- Текст великий, цифрова клавіатура для number.
- «Перевірити» → правильно/помилка/розкриття після 3 спроб.
- Прогрес-бар сесії, можливість вийти з підтвердженням.

### 5.6. Підказки

- 2 рівні + повне рішення.
- 1-ша спроба без підказки = 3⭐. 2-га без / 1 підказка = 2⭐. 2 підказки = 1⭐. Розкриття = 0⭐.

### 5.7. Робота над помилками

Інтервальне повторення: 1, 2, 4, 8, 16 днів. 5 успіхів → видалення.

### 5.8. Гейміфікація

- **Зірки:** 3/2/1/0 за задачу.
- **XP:** 10 xp за зірку. Рівень `√(xp/100)+1`.
- **Монети:** 5 за сесію + 1 за ⭐.
- **Аватар:** рівні + аксесуари.
- **Ачивки:** «Тиждень підряд», «Блискавка», «Перфектна сесія» тощо.
- **Streak:** +1 день за виконане щоденне. «Щит» = 50 монет.

### 5.9. Адаптивна складність

3 правильні поспіль → +1 рівень. 2 помилки в межах 5 → −1 рівень. Ручне перевизначення доступне.

### 5.10. Карта світу

Тематична візуалізація прогресу, відкриття локацій по дереву залежностей.

### 5.11. Календар

Heat-map з днями активності, тап → деталі дня.

### 5.12. Профіль

Аватар + рівень + статистика + ачивки + аксесуари.

### 5.13. Налаштування

Звуки, сетинг, ціль, ім'я, батьківська панель, скидання.

### 5.14. Батьківська панель

Вхід через PIN (SHA-256). 5 вкладок: зведення, по темах, календар, журнал помилок, налаштування.

### 5.15. Звуки

4 ефекти: правильно / помилка / кінець сесії / новий рівень. Web Audio API без файлів.

## 7. Технічна архітектура

### Стек

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (з persist)
- React Router
- Framer Motion (v0.5+)
- vite-plugin-pwa
- localStorage → IndexedDB → Cloudflare D1
- Vitest + Testing Library

### Шари

```
UI (React) → Domain (use-cases) → Generators → Storage (адаптер)
```

### Модель даних

Див. `src/store/profile.ts` — `Profile`, `TopicProgress`, типи.
