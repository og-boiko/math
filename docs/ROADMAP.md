## Метрики успіху

**Для тебе:**

- Артем заходить ≥ 5 днів на тиждень
- ≥ 80 задач/тиждень
- Точність зростає у часі
- Перші 2 місяці 5 кл. — без сюрпризів

**Для проекту:**

- Lighthouse mobile ≥ 90
- Bundle ≤ 250 KB gzip
- 0 критичних помилок
- Старт PWA ≤ 1с

## Ризики

| Ризик                            | Мітигація                                         |
| -------------------------------- | ------------------------------------------------- |
| Нудно за тиждень                 | Зміна тем + ачивки + зміна сетингу                |
| Складність не вгадана            | Адаптив + батьківська панель                      |
| Дивні задачі від генератора      | Юніт-тести з санітарними перевірками              |
| Фарм легких задач                | Бонус-зірки за прогрес-рівень, не за кількість    |
| Втрата прогресу                  | Експорт JSON + майбутній D1                       |
